## 어려웠던 점 / 배운 점

- Git 초기 설정(user.name, user.email)이 없으면 커밋 자체가 안 된다는 것을 경험으로 배웠다.
- GitHub Pages 배포를 위해 `main` 브랜치와 원격 저장소 연결 흐름을 직접 익혔다.
- 빈 저장소에서 `git add` 할 파일이 없으면 커밋이 생성되지 않아 push가 실패한다는 점을 알게 됐다.
